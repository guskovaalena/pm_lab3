\doxysubsection{Подробное описание}
Генератор, улучшающий классический метод середины квадрата. 

На каждом шаге:\+ state = state\texorpdfstring{$^\wedge$}{\string^}2, берутся средние 32 бита, циклический сдвиг, XOR с предыдущим состоянием. Предотвращается вырождение в ноль.
